Task 3: 10 points
Deadline 26 April before practice class. 

Write a program (GUI based) which does the same job as Notepad (Windows' application). However, our application is very simple. 

It must have a menu Bar and two menus File menu and Help menu. Inside the File menu, there should be 3 functions, Open, Save and Close
1. Open, which lets user to chose a text file to display in the text area. 
2. Save, should save the contents of text area in a text file. 
3. Close, should terminate the application. 

Help Menu: upon clicking it should generate a message box to give brief info. e.g. version of the applicaiton, probably auther name....
